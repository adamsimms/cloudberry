# GoPro HERO5 support

**Status: not planned**

Cloudberry v3 targets the Pinchard's Island field rig: **GoPro HERO3**, **HERO4**, and the **Pi camera module** (`picamera2`).

HERO5 uses a different Wi‑Fi API and control stack than H3/H4. Adding support would require separate research, hardware testing, and new code paths in `cloudberry/gopro.py` — outside the scope of the island deployment.

If you need HERO5 support in the future, open a new issue with API notes or a prototype PR.
