# Pi Camera Suite (legacy field note)

**Cloudberry v3.0** uses **picamera2** for Pi camera capture. Install on the Pi:

```bash
sudo apt install -y python3-picamera2
```

Or use `./setup.sh` from the repo root (installs base deps; add picamera2 on Pi hardware).

For the full rig, prefer the `cloudberry` CLI and `config.ini` (`camera_type = picamera`) rather than the old Camera Suite GUI workflow below.

---

## Legacy: Camera Suite (historical)

The following steps were used when Camera Suite drove capture before v3.0. Kept for reference only.

### Download and install

```bash
cd ~
git clone https://github.com/scottlambert7366/picamera.git camerasuite
cd camerasuite
./camerasuite.sh -platform xcb
```

### Autostart (legacy)

```bash
mkdir -p ~/.config/autostart
cp ~/camerasuite/camerasuite.desktop ~/.config/autostart/
```

### Disable autostart

```bash
rm ~/.config/autostart/camerasuite.desktop
```
